# Gira++ - Frequently Asked Questions (SEO-Optimized)

## What is Gira++?

**Gira++** is a privacy-focused, open-source Android app for Lisbon's GIRA bike-sharing service. It's a faster, lighter, and more private alternative to the official Gira app.

## How is Gira++ different from the official Gira app?

Gira++ offers several key advantages:
- **100% Privacy**: Zero tracking, no analytics, no third-party connections
- **9 Languages**: English, Portuguese, Spanish, French, German, Italian, Arabic, Russian, Hindi
- **Lightweight**: Only 17MB vs. larger official apps
- **Faster Performance**: Instant station loading with real-time WebSocket updates
- **Open Source**: Fully transparent code you can verify yourself

## Is Gira++ safe to use?

Yes! Gira++ is completely safe:
- Open-source code that anyone can audit
- No tracking or data collection
- Only communicates with official EMEL/Gira API
- MIT licensed
- Community-reviewed on GitHub

## How do I download Gira++?

1. Visit the [GitHub Releases page](https://github.com/seiflesbat/gira-plus/releases/latest)
2. Download the latest APK file
3. Install on your Android device (Android 7.0+)
4. Grant necessary permissions (location, etc.)
5. Start using Gira bikes with complete privacy!

## Does Gira++ work with real GIRA bikes in Lisbon?

Yes! Gira++ uses the **official EMEL/Gira API**, so it works perfectly with all real GIRA bike stations across Lisbon. It's a client app that connects to the same backend as the official app.

## Is Gira++ free?

Yes, **Gira++ is 100% free** and always will be. It's an open-source project licensed under MIT, meaning:
- No advertisements
- No in-app purchases
- No subscription fees
- No premium features
- Completely free forever

## What Android versions does Gira++ support?

Gira++ supports **Android 7.0 (API 24) and above**. This covers approximately 95% of Android devices currently in use.

## How does Gira++ protect my privacy?

Gira++ protects your privacy through:
- **No Analytics SDKs**: Zero tracking libraries
- **No Third-Party Services**: Only connects to official Gira API
- **No Data Collection**: We don't collect any user data
- **Local Storage Only**: All data stays on your device
- **Open Source**: Verify the code yourself on GitHub

## Can I use Gira++ for electric bikes (e-bikes)?

Yes! Gira++ supports **both classic and electric GIRA bikes**. The app shows which stations have electric bikes available.

## Does Gira++ require my location?

Yes, to show nearby bike stations and provide optimal functionality, Gira++ needs location permission. However, **your location is never sent to third parties** - it's only used locally on your device and sent to the official Gira API when needed.

## What permissions does Gira++ require?

Gira++ requires minimal permissions:
- **Location**: To show nearby bike stations
- **Internet**: To communicate with Gira API
- **Storage**: To save app settings and trip history locally

**No invasive permissions** like camera, contacts, or microphone.

## Can I use Gira++ without internet?

Limited functionality is available offline:
- View previously loaded map areas
- Access trip history
- See app settings

Real-time features (bike availability, unlocking bikes) require internet connection.
